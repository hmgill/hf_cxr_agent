# CXR Agent

A chat interface for chest-X-ray interpretation. Attach a CXR and ask for a read —
the agent triages the image, reasons over it with **NV-Reason-CXR-3B**, localizes
findings with **NV-LocateAnything-3B**n, and can render an **interactive overlay** in the right-hand panel. Turn
on the voice option for a spoken summary.

**Decision support only — not a diagnostic device.** Always confirm findings with a
qualified radiologist.
